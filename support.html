快速开始

  • 点击“开始取色”，从屏幕任意位置选择颜色。
  • 点击“上传图片”，选择本地图片后点击像素取色。
  • 使用 Option + Command + C 快速启动系统取色器。
  • 点击格式右侧的“复制”按钮复制颜色或开发代码。

常见问题

是否需要联网？
不需要。取色、图片分析、历史和收藏均在本地处理。

为什么关闭窗口后应用仍在运行？
取色器是菜单栏常驻工具，可点击菜单栏“取色”重新打开。

如何反馈问题？
请通过 GitHub Issues 联系开发者。

English

Color Picker is an offline macOS color sampling tool. Use Start Picking or Option + Command + C for screen sampling, and Upload Image for local image sampling. For support, open a GitHub Issue.
