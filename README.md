# 取色器隐私政策

这是取色器的公开隐私政策与支持静态站点，页面不包含 JavaScript、Cookie、统计、跟踪或第三方字体。

发布地址：<https://linnasean.github.io/ai-color-picker-privacy/>

支持地址：<https://linnasean.github.io/ai-color-picker-privacy/support.html>

## GitHub Pages

仓库设为公开后，在 **Settings → Pages** 中选择默认分支和 `/ (root)` 目录发布。

## 本地预览

```bash
python3 -m http.server 8765 --directory .
```

然后打开 <http://127.0.0.1:8765/>。
